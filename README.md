**📘 Social Profile API (Golang)**

A simple Social Profile Management API built with Golang that supports:

1. 🔐 Google OAuth2 Login

2. 🔐 Microsoft OAuth2 Login

3. 🪪 JWT Authentication

4. 👤 Create / Update Profile

5. 🖼 Upload Profile Image

6. 🏗 Clean Modular Architecture

**This project is designed to demonstrate OAuth integration, JWT authentication, REST API design, and modular Go architecture.**

🚀 Tech Stack

* Go (Golang)

* OAuth2 (Google & Microsoft)

* JWT Authentication

* net/http

* Clean Folder Structure
